# Disability Confident Portal — prototype

A static, self-contained HTML/CSS/JS click-through prototype of the Disability Confident Portal, covering:

- Recruitment (advert checker, interview checker)
- Onboarding (starter list, dashboard, inclusion passport, manager conversation coach)
- Adjustments (request, decision guide, escalation, agreed record)
- Employee support chatbot journey

Styled in GOV.UK Design System conventions, recoloured to Scope's brand palette.

## View it live

Once GitHub Pages is enabled for this repo (Settings → Pages → Deploy from branch → main → /root), it will be live at:

`https://<your-username>.github.io/<repo-name>/`

## Run it locally

No build step needed — just open `index.html` in any browser.
